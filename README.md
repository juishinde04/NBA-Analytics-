# NBA-Analytics-
Project Overview

Built an end-to-end data analysis pipeline using NBA player performance data to extract insights, visualize trends, and predict scoring performance.

⚙️ Tech Stack

Python (Pandas, NumPy)

SQL (SQLite)

Matplotlib, Seaborn

Scikit-learn

🔄 Workflow
1. Data Collection

Initially attempted API-based data extraction

Switched to Kaggle dataset due to API limitations

2. Data Cleaning

Removed irrelevant columns (NICKNAME, COMMENT)

Filtered out players who did not play

Converted time and statistical columns to numeric

Handled missing values and duplicates

3. Exploratory Data Analysis (EDA)

Identified top scorers, playmakers, and rebounders

Analyzed relationships between minutes played and scoring

Visualized performance distributions and trends

4. Machine Learning

Built a Linear Regression model to predict player points

Achieved Mean Squared Error ≈ 29.75

5. Visualization

Created dashboard-style visualizations using Python

Highlighted key relationships and performance patterns

📊 Key Insights

Players with higher minutes tend to score more points

Positive relationship observed between assists and scoring

Player performance varies significantly across games
